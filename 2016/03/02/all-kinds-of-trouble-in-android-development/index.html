StatusBar 与 ActionBar 显示问题

使用 windowTranslucentStatus 不显示 ActionBar 时容易出现的坑:

  • toolbar 错位到statusbar底部
  • toolbar back icon button 和 title 不在同一水平线上

  1. toolbar 错位到statusbar底部
    错误：
