打开华为 Android 手机 DEBUG 日志

  • Android
  • Debug
  • Log
技术

目录

  1. 1. 解决方法：
  2. 2. 参考

最近使用华为安卓真机调试 App 时发现 App 里的日志不完整， DEBUG 和 VERBOSE 级别的日志都不能显示，只有 INFO
级别以上日志才能正常输出，刚开始以为是 Android Studio 配置和手机的问题，在进行了修改之后发现还是不行。
最终在网上看到了相关的文章，是手机把Log关掉了的原因，在此记录下操作步骤。

解决方法：

自己的华为荣耀 7 和公司的测试机 华为 P6 都进行了设置，最终都可以正常输出日志了。

  1. 进入手机工程模式
    打开手机拨号界面，然后输入：

这个时候可以看到工程菜单

  1. 然后选择后台设置进入后台设置

  2. 打开 Log
    点击LOG设置，选择LOG 打开，或者选择AP日志。

  3. 测试
    回到要调试的应用，查看 Android Studio 里的 Android Monitor应该可以看到输出的 DEBUG 日志了，
    如果看不到可能需要重启手机再试。
