在小米路由器 mini 中安装 Transmission 下载器

  • OpenWrt
  • Transmission
  • 小米路由器
技术

目录

  1. 1. 刷入开发版
  2. 2. 刷入SSH
  3. 3. 安装 Transmission 插件
    1. 3.1. 更改OpenWrt源
  4. 4. 参考

最近入手了一个小米的路由器 mini，一直对其挂载硬盘比较感兴趣，抽空折腾了一下，为以后挂载 bt 下载器做准备。

刷入开发版

  1. 下载开发版 ROM
    下载最新开发版本，打开网页后选择 ROM 切换到 ROM 下载：

  2. 刷入开发板
    查看刷机教程

    首先，请先准备一个U盘，并确保这个U盘的格式为FAT或FAT32.
    接下来，就是具体的操作流程了。

    1. 在miwifi.com官网下载路由器对应的ROM包，并将其放在U盘的根目录下，命名为miwifi.bin
    2. 断开小米路由器mini的电源，将U盘插入路由器的USB接口
    3. 按下reset按钮后重新接入电源，待指示灯变为黄色闪烁状态后松开reset键
    4. 等待5~8分钟，刷机完成之后系统会自动重启并进入正常的启动状态（指示灯由黄灯常亮变为蓝灯常亮），此时，说明刷机成功完成！

    如果出现异常/失败/U盘无法读取状态，会进入红灯状态，建议重试或更换U盘再试。

刷入SSH

  1. 下载 ssh 固件
    下载SSH 固件，打开网页后，提示需要先进行登录，登录完成后才可以进入，之后点击『下载工具包』即可。

  2. 刷入 ssh 固件

    1. 请将下载的工具包bin文件复制到U盘（FAT/FAT32格式）的根目录下，保证文件名为miwifi_ssh.bin；
    2. 断开小米路由器的电源，将U盘插入USB接口；
    3. 按住reset按钮之后重新接入电源，指示灯变为黄色闪烁状态即可松开reset键；
    4. 等待3-5秒后安装完成之后，小米路由器会自动重启，之后您就可以尽情折腾啦 ：）

安装 Transmission 插件

这里主要参考此贴路由mini安装OpenWRT源的Transmission插件，实现PT下载（需SSH）。但是楼主提供的修改OpenWRT源地址在国内很慢，想着国内会不会有镜像源，Google 了一下，找到了科大的镜像源。

更改OpenWrt源

修改帖子中提供的 OpenWRT 源地址：http://downloads.openwrt.org/ 为 http://mirrors.ustc.edu.cn/openwrt， 其中科大的镜像源是每次同步最新的一个版本，搜索时可能本文所列的已经不是最新的版本了。
例如 barrier_breaker 对应 14.07 。
这里只要对应上 cpu 架构：ramips和型号：mt7620a即可。
例如这里 14.07 版本的地址为： http://mirrors.ustc.edu.cn/openwrt/barrier_breaker/14.07/ramips/mt7620a/packages/base
